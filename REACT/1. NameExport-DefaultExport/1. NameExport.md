# Named Export và Default Export

### Module
Module là một <b>file chứa mã JS</b> mà bạn có thể <b>tái sử dụng</b> ở các file khác.  
⌯⌲ <b>export</b> được sử dụng để chia sẻ hàm/ biến/ class từ một <b>module</b>.  
---
